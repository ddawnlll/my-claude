# Kayrae Anti-Clone Policy

> What to copy vs what to NEVER copy from reference sites (giydiriyo.com, Kayrae, etc).

## The Distinction

```
GOAL:     Achieve Kayrae-LEVEL quality, not produce a Kayrae CLONE.
MEANS:    Copy the visual grammar. Never copy the identity.
RESULT:   A site that feels equally premium but is unmistakably its own brand.
```

## ✅ COPY (Visual Grammar)

### Structural Patterns
```
✅ Background layer architecture (5-7 layers)
✅ Border opacity scale (6% / 10% / 18% / 25%)
✅ Glass morphism with backdrop-blur + solid fallback
✅ Card surface hierarchy (default → glass → interactive)
✅ Section spacing rhythm (variable, not uniform)
✅ 620px mobile breakpoint
✅ Clamp() fluid typography
✅ [&_em]:font-serif editorial emphasis PATTERN (with different serif font)
```

### Motion Patterns
```
✅ One cinematic moment per section PHILOSOPHY
✅ Blur→clear reveal TECHNIQUE (filter animation)
✅ CSS grid-template-rows accordion TECHNIQUE
✅ clip-path connector reveal TECHNIQUE
✅ CSS offset-path border beam TECHNIQUE
✅ 0.18s stagger RHYTHM
✅ cubic-bezier(0.22,1,0.36,1) EASING
```

### Atmosphere Patterns
```
✅ SVG feTurbulence grain noise TECHNIQUE
✅ CSS radial-gradient dot grid PATTERN
✅ Layered radial gradients + mix-blend-overlay APPROACH
✅ Mask-faded edges on patterns TECHNIQUE
✅ CSS-only (zero JS) atmosphere PHILOSOPHY
```

### Hover Patterns
```
✅ Translate-y instead of scale APPROACH
✅ Targeted transitions (never transition-all) RULE
✅ Border accent on hover TECHNIQUE
✅ Inner element reveal on hover PATTERN
✅ 150-200ms hover timing RANGE
```

## ❌ NEVER COPY (Identity)

### Brand Identity
```
❌ Exact color palette (copper #C86B3C — that's Kayrae's identity)
❌ Logo or logotype
❌ Brand name or product name
❌ Taglines or marketing copy
❌ Exact typography stack (Instrument Serif — license-restricted)
   → Use SIMILAR QUALITY alternative (Playfair Display, Cormorant, etc.)
```

### Content & Structure
```
❌ Exact section order
❌ Exact section content (text, headings, descriptions)
❌ Exact number of sections
❌ Exact component geometry (specific card sizes, grid column counts)
❌ Exact illustrations, icons, or screenshots
❌ Exact canvas/graph/product visualization layouts
```

### Specific UI Elements
```
❌ Exact navbar structure (link count, CTA placement — adapt to YOUR content)
❌ Exact footer columns (adapt to YOUR information architecture)
❌ Exact mobile menu design (adapt to YOUR navigation depth)
❌ Exact CTA panel dimensions and text
❌ Exact testimonial layout and count
```

## 🎨 ADAPT (Make It Yours)

### Color Adaptation

```
Kayrae (reference):      Your Brand (example — Gebze Fiber):
─────────────────────────────────────────────────
Copper #C86B3C           → Cyan #00A3E0 (fiber optic metaphor)
Gold #FFC382              → Cyan glow #00D4FF
Sage green #8DD07A        → WhatsApp green #25D366
Warm white #FFFAF5        → Neutral white #EDEDED

SAME: True-black stack #050505 → #080808 → #0A0A0A
SAME: Border opacity scale 6%/10%/18%/25%
```

### Typography Adaptation

```
Kayrae (reference):      Your Brand:
─────────────────────────────────
Inter (UI)               → Geist Sans (excellent TR support)
Instrument Serif (editorial) → Playfair Display (Latin-ext, TR chars)
Geist Mono (technical)   → Geist Mono (keep — it's already good)

SAME: [&_em]:font-serif pattern
SAME: 3-font editorial-tech stack
DIFFERENT: Specific font families
```

### Content Adaptation

```
Kayrae:  AI product visualization SaaS
         → Canvas/graph metaphors, enterprise tone, English

You:     [Your product/service]
         → [Your industry metaphors], [Your audience tone], [Your language]

SAME:     Section composition philosophy (hero→trust→features→process→CTA)
SAME:     Premium visual quality
DIFFERENT: Content type, tone, language, metaphors
```

## The Litmus Test

After building, ask:

```
1. Does this look like it COULD be a Kayrae product page?
   → If YES: you cloned too much. Change color, type, layout.

2. Does this feel equally PREMIUM as Kayrae?
   → If NO: you didn't copy enough grammar. Add depth, motion, hover.

3. Can I tell what BRAND this is within 3 seconds?
   → If NO: the brand identity is too weak. Strengthen color, type, voice.

4. Would someone confuse this with THE reference site?
   → If YES: IMMEDIATE FAIL. Go back and change visual identity.
```

## Implementation Rules

```
1. NEVER copy-paste CSS values directly from reference sites.
   Understand the TECHNIQUE, then write your own implementation.

2. NEVER screenshot reference sites for "pixel-perfect" matching.
   Reference the QUALITY LEVEL, not the exact pixels.

3. ALWAYS adapt the color palette to the brand.
   The technique (5-layer background) is copyable.
   The color (#C86B3C) is not.

4. ALWAYS adapt typography to the content language.
   Instrument Serif doesn't have Turkish characters.
   Playfair Display does. Use that.

5. ALWAYS adapt section content to the product.
   Kayrae sells AI SaaS. Gebze Fiber sells cable installation.
   Same quality grammar. Completely different content.
```

## Quality Without Cloning

```
The goal is NOT to make sites that look like giydiriyo.com.
The goal is to make sites that feel AS PREMIUM AS giydiriyo.com.

Premium feel comes from:
  ✅ Visual grammar (depth, motion, hover, surface)
  ✅ Quality standards (no generic patterns)
  ✅ Intentionality (every choice has a reason)

Premium feel does NOT come from:
  ❌ Copying someone else's exact design
  ❌ Using the same colors
  ❌ Matching pixel positions
```
